v.0.1.4007 29.01.12

[GUI]
* Иконки в EMI подсвечиваются зеленым при наведении
* При начале новой игры можно выставить несколько переключателей, для выбора типа и уровня сложности игры

[Gameplay]
* Уровни стали конечными. Для прохождения каждого уровня необходимо выполнить не менее одного заранее заданного условия:
** Уничтожить заданное количество любых кораблей
** Уничтожить заданное количество кораблей определенного типа
** Пролететь заданное расстояние
* Появился тип игры "Random", при выборе которого будет создан и запущен уровень со случайными условиями для победы
* После каждого уровня выводится статистика
